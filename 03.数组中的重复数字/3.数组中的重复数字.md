# 3. 数组中的重复数字

## 题目

在一个长度为n的数组里的所有数字都在0到n-1的范围内。 数组中某些数字是重复的，但不知道有几个数字是重复的。
也不知道每个数字重复几次。请找出数组中**任意一个重复**的数字。 
例如，如果输入长度为7的数组{2,3,1,0,2,5,3}，那么对应的输出是第一个重复的数字2。

## 思路

思路1. 以空间换时间，使用hash
思路2. 因为这些数字都是小于n的，所以可以使用数组的下标进行定位，数字与坐标是一一对应的
		1. 从左至右遍历
		2. 如果该位置数字等于下标，下标则向前移动
		3. 否则的话，则根据此数字，去寻找该位置的数字
			a. 如果相等（说明此位置已有正确的数字），该数字重复
			b. 如果不等，两元素交换位置

## 反思

本题解法适合：返回任意一个重复数字

如果要获得全部重复数字（并且没重复），该方法不适用

## 题目2

n+1 长度的数组，元素在 1 ―― n范围内，一定存在重复数字，找出任意一个重复数字，不能修改数组！

## 思路

如果使用一个辅助数组的话，时间复杂度为O(n)，空间复杂度O(n)，这个思路比较好写；

如果不使用辅助数组，将值的范围每次缩短1/2，然后搜索这个范围内元素出现的次数，需要搜索lg(n)次，每次O(n),时间复杂度O(nlgn),空间复杂度O(1)

## 反思

二分查找，非递归写法是如何控制边界的

	start = mid + 1
	end = mid - 1
	return mid
